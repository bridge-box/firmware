# BridgeBox Firmware

Open-source прошивка для plug-and-play сетевого моста с mesh-сетью и удалённым управлением через Tailscale/Headscale.

## Что это

BridgeBox — это прошивка для NanoPi R3S (OpenWrt), которая превращает компактный одноплатник в готовый mesh-роутер:

- **Plug & play** — воткнул кабель, коробка получила интернет
- **Mesh-сеть** — автоматическое подключение к Headscale/Tailscale
- **Удалённый доступ** — SOCKS5-прокси через mesh для доступа к локальной сети
